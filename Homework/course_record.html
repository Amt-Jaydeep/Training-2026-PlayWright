課程介紹

數據分析、程式開發者必備：Python＋SQL 語法

根據全世界公認最大的資料科學社群 Kaggle 在 2021 Kaggle Data Science & Machine Learn Survey 中，Python 和 SQL 於「Most Popular Programming in 2021」分占前兩名，可見兩者的重要性不可言喻。

Python 結合 SQL

Python 是熱門的程式語言，用途廣闊，常用於網路爬蟲、數據分析或網站開發，基本上都會有大量的數據產生。有數據產生，必然得搭配 SQL 資料庫。而時下最夯的 AI 產品也主要都使用 Python 開發，其中所使用的資料即可儲存於資料庫內，因此兩者的結合運用勢在必行。

SQL能做什麼？

SQL (Structured Query Language) 是能夠查詢與操作資料庫的語言，用來對資料庫進行儲存、操作與檢索的動作。不管你是為了找尋資料還是處理數據，SQL 就是那一個能夠協助我們快速找到有用訊息的工具。透過它，我們可以更有效率的分析龐雜的原始資料，不論你想要做數據分析應用程式、手機APP或架設動態網站系統等，都必須學習SQL資料庫。

但資料庫的資料須要透過程式語言來存取，開發系統功能給使用者操作，進而資料分析，所以需要瞭解程式語言與資料庫之間的關係。

這堂課就是教學 Python 串接 SQL 資料庫語法的作法，

● 學成後 ● 你可以

  • 直接透過資料庫提取資料，輕鬆整合且運用資料，進而商業分析
  • 製作出人人都可輕易操作的查詢介面
  • 大幅度的開拓全方位且多角化的工作能力
